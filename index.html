What Makes It Duff

The goal here is not to be everything for everyone. Duff Linux leans into a specific setup: modern Plasma, sensible storage defaults, and less time spent rebuilding a desktop from scratch.

01 A desktop that starts finished

Plasma is the focus, not an afterthought. The live system, theming choices, and package set are all biased toward getting you into a comfortable daily driver quickly.

02 Safer updates with snapshots

BTRFS is paired with automatic snapshot handling so package changes are less nerve-racking and rollback workflows stay within reach for real systems.

03 Still recognizably Void

Under the polish, Duff Linux stays grounded in Void Linux tooling and packaging, which keeps the distro fast, understandable, and appealing to people who actually like their base system.
