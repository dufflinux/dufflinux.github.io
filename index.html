Duff Linux

An opinionated distro based off dani-77's d77void Linux distribution, focused on KDE Plasma with BTRFS.🍺

Download Latest ISO View Source on GitHub
Desktop KDE Plasma

Latest available Plasma stack with a lightly-themed, ready-to-use desktop feel.

Filesystem BTRFS + Snapshots

Automatic pre-transaction snapshots and regular backup-friendly behavior out of the box.

Installer Calamares Live ISO

Boot a live environment, install graphically, and land in a usable system quickly.

Packages XBPS + Flatpak

OctoXBPS for native packages, update notifications, and Discover-backed Flatpak support.

Quick Overview

The current project focus is clear and practical: install fast, recover safely, and keep the platform approachable.

1
Kernel variants available

Images are built around multiple kernel tracks, including current options highlighted in the main project README.

2
Live environment first

Boot into a usable session, launch Calamares, and install from a system that already reflects the desktop vision.

3
Package management without friction

OctoXBPS and OctoXBPS Notifier help native package management feel less bare while keeping Void’s native stack intact.

4
Built for people who tinker

The public repo includes ISO build scripts and helper tooling, so the distro is not a black box if you want to inspect or rebuild it.

Build Your Own ISO

The website does not replace the wiki, but it should make the repo feel approachable. If you want to rebuild Duff Linux yourself, the repository already exposes the key entry points.

Step 1 Prepare the build environment

Start with the setup helper so the host machine has the expected dependencies for ISO generation.

scripts/setup-iso-build-env.sh
Step 2 Use the helper build scripts

Predefined scripts exist for common image variants, including AMD and NVIDIA oriented builds.

scripts/build-iso.sh
Step 3 Dive into the build logic

The main `duff` build script and `build/` tree show how the live image, installer, and defaults are assembled.

duff
Step 4 Follow the full guide

The project wiki is the place for the complete ISO generation workflow and supporting documentation.
